 Amazon Sales Data Analysis

This project analyzes Amazon sales data using SQL queries and an interactive Excel dashboard.

## Tools Used
- SQL (MySQL)
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Key Insights
- Sales increased significantly in 2025 compared to previous years.
- Vendor LCO generated the highest revenue among vendors.
- California and Texas contributed the highest revenue among states.
- Sales peak during the last months of the year, especially November and December.
- The average order value is around 80–90 per order.
